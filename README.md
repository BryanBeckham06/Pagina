# Minha Pagina
### Informações Pessoais


# Linguagens:
- Node.js
- React Native
- Yarn
- Expo

# Ambiente:
- Necessário ter instalado: - VsCode - NodeJS

# Como executar:
- 1 Clone esse repositório
- 2 Abra com VsCode
- 3 Abra o terminal e acesse a pasta
   ```
  cd Mypage
  ```
- 4 Instale as dependências
  ```
  yarn
  ```
- 5 Execute o app via web
  ```
  yarn web
  ```
